# ghost_maze
2d maze game

# How to play

 Key press | Action
 ----------|--------
 A|move left
 D|move right
 W|move up
 S|move down
 J|buy 1 strength per 10 gold
 H|buy 5 health per 10 gold
##goal
get out of the maze
